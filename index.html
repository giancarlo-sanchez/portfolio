My Sneakers

E-commerce website built in Express Node.js for backend and React, Redux, CSS3 for frontend.

Main takeaways of the project:

  • The pages are defined as React function components and the state and dispatched actions are handled by Redux hooks creating a more intuitive and readable code.
  • Created a shopping cart component by utilizing a slice of state that stores users choices(array of objects), shipping and payment info, creating a checkout flow that can only be completed when we have the three parts of the slice, preventing users from posting incomplete orders to the backend and causing the app to crash.
  • Designed a catalog page using CSS Flexbox getting responsiveness across devices.

TAPPDIN

Social media app inspired on Untappd and built using Express Node.js for backend and frontend, JavaScript, HTML5 and PUG template engine.

Main takeaways of the project:

  • Implemented Pug template engine to render dynamic content and creating reusable HTML components that are deployed across multiple views on the site using Pug mixin reducing the amount of code drastically.
  • Incorporated JSON Web Token (JWT) to handle user authentication across sessions by storing the JWT in a cookie in the browser. The token is send along with backend server request providing a fast authentication method and preventing other users from accessing private data.
  • Designed a relational database that is queried by the backend through the Sequelize ORM for Postgres avoiding SQL query complexity and increasing the app scalability.
